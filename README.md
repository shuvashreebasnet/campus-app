**Objective:**
To make finding out about and joining campus involvement opportunities like events and clubs convenient for students.

**To run the application:**
Open the campus-app folder in VS code. Click the run tab above. Click start debugging. Select Chome as your web debugger. 
The application will immediately open in a separate browser tab!
